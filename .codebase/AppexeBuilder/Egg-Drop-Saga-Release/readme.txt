Welcome to Egg Drop Saga Setup

Thank you for installing Egg Drop Saga!

Egg Drop Saga is a fun and lightweight arcade-style game where your goal is to catch falling eggs and score as many points as possible.

Before you begin installation, please make sure:
- You are running Windows 10 or later
- No previous version of Egg Drop Saga is currently running

This setup will:
- Install the game to your system
- Copy all required game files and libraries
- Create Start Menu and Desktop shortcuts

Click "Next" to continue the installation.
